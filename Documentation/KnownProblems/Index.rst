==============
Known Problems
==============

: Only accepted terms are available, `pending types and properties <https://pending.schema.org/>`__ are not available
(yet). If they are integrated into the core vocabulary, they are available within the next update.

Have a look at the `issue tracker <https://github.com/brotkrueml/schema/issues>`__ to see which features and bugs are
already opened. If you encounter any problems, you can open an issue and preferably file a pull request.
